# Code of Conduct

## Our Standards

FieldStay is a private repository with a small, focused team. We expect all contributors and collaborators to:

- Communicate directly and professionally
- Give and receive technical feedback constructively
- Respect the time and context of others when reviewing code or filing issues
- Assume good intent in ambiguous situations before escalating

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language in any communication channel connected to this project (GitHub issues, PRs, Slack, email)
- Deliberately introducing security vulnerabilities or data integrity risks
- Sharing access credentials or repository contents outside of authorized team members

## Reporting

If you experience or witness behavior that violates these standards, contact the project owner directly at **hello@fieldstay.app**.

Reports will be handled confidentially and acted on promptly.
