# Setting Up Your First Property

After OwnerRez sync completes, a few quick settings ensure turnovers generate
correctly and your crew has everything they need.

## What the Sync Handles Automatically
Property name, address, coordinates, bedrooms, bathrooms, max guest count, WiFi
credentials, check-in instructions, house manual, amenity flags, check-in and
checkout times.

## What You Need to Set Up Manually

### Cleaning Cost
Go to Properties → [Property Name] → Settings and enter:
- Standard cleaning cost (flat fee per turnover)
- Same-day premium percentage (markup for same-day checkout/checkin)

### Default Crew Member
Go to Properties → [Property Name] → Crew and select the crew member who most
frequently cleans this property.

### Guest Guidebook
Go to Guidebook → [Property Name] and review WiFi, check-in instructions, and
house rules. Toggle Published when ready — guests won't see the guidebook until
it's published.

## Setup Checklist
- Cleaning cost and same-day premium set
- Default crew assigned
- Guidebook reviewed and published
- Inventory template applied (optional)
